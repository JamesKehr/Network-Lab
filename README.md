# Network-Lab

Docs on how to setup various network lab servers. These are non-production configurations meant for learning and testing.

## Available Documentation

### GEARS-GW Ubuntu Gateway Setup

Complete documentation for setting up an Ubuntu-based network gateway with support for IPv4, IPv6, NAT, DHCP, DNS64, and NAT64.

**[Start with the GEARS-GW Index](GEARS-GW-Index.md)** for an overview and navigation to specific setup guides:

- [Basic Setup](GEARS-GW-Basic-Setup.md) - Ubuntu installation, networking, NAT4, and tcgui
- [DHCPv4 Setup](GEARS-GW-DHCPv4.md) - IPv4 DHCP configuration
- [IPv6 Setup](GEARS-GW-IPv6-Setup.md) - DHCPv6, radvd, and BIND9 DNS64
- [NAT64 Setup](GEARS-GW-NAT64.md) - IPv6-to-IPv4 translation with jool
- [IPv6 NCSI Responder](GEARS-GW-NCSI-Responder.md) - Internal connectivity detection
- [IPv6-Only Configuration](GEARS-GW-IPv6-Only.md) - Switch between IPv6-only and dual-stack
- [Certbot Setup](GEARS-GW-Certbot.md) - DNS over HTTPS certificates (future)
